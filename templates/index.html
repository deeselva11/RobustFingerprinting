 WELCOME   TO   THE   Application

Database fingerprinting has been widely adopted to prevent unauthorized sharing of data and identify the source of data leakages. Although existing schemes are robust against common attacks, like random bit flipping and subset attack, their robustness degrades significantly if attackers utilize the inherent correlations among database entries.

In this app, you can perform correlation attacks on the Fingerprinted census database. Fingerprint algorithm is not much efficient against correlation attack. Malicious SP can use the correlation attack and cause database owner to accuse innocent sp. In this Application we are using robust fingerprinting algorithm as a migigation.
